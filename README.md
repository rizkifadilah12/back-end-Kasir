# back-end-Kasir
Back-End kasir with JsonPlaceholder <br>

Step 1
install json placeholder 
<h2> npm install -g json-server</h2>

Step 2
Run Json-Server
<h3>json-server --watch db.json --port 3004</h3>
<b>NOTE:</b>
--port (Custom Port)
<h5>Access : http://localhost:3004/ OR http://localhost:{CustomPort}/</h5>
